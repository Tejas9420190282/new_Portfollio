Quick reference — what to customise

File	                            What to edit

1) src/App.jsx → EXPERIENCES	    Your job title, company, responsibilities, tech stack
2) src/App.jsx → PROJECTS	        Project names, descriptions, GitHub/live links, images
3) src/App.jsx → SKILLS	            Add / remove skill categories and tags
4) src/App.jsx → EDUCATION	        Your degree, institution, CGPA
5) src/App.jsx → CERTIFICATIONS	    Your certificates
6) src/App.jsx → Hero h1	        Replace "Abhishek Singh" with your name
7) src/App.jsx → Nav logo	        Change <AS /> to your initials
8) Contact links	                Update email, GitHub, LinkedIn URLs
